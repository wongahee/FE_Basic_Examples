## 📋 프로젝트 개요
웹 브라우저의 **데이터 저장, 동적 UI, 사용자 인터랙션** 등 프론트엔드 핵심 기능들을 학습하며 직접 구현한 프로젝트입니다.

<br>

## 🛠 기술 스택

### Frontend
- HTML5, CSS3, JavaScript (ES6)

### UI Framework
- Bootstrap 4.6.0
